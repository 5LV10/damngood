# DamnGood

One source of truth for AI coding tool configuration.

DamnGood centralizes:

- MCP servers
- global agent instructions
- reusable skills

Then syncs to supported clients with safe defaults.

## Why teams adopt it

- no more copy-paste config drift across tools
- shared standards with per-client assignment controls
- safe syncing that preserves user-owned content
- works well for solo devs and team onboarding
